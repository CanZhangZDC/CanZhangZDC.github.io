CSS3 动画

2.学习参考：

https://www.runoob.com/css3/css3-animations.html

@keyframes规则：https://www.runoob.com/cssref/css3-pr-animation-keyframes.html

3.感悟&总结

CSS3动画可以取代网页动画图像，Flash动画，和JAVAScripts。

@keyframes规则；
当在 @keyframes 创建动画，把它绑定到一个选择器，否则动画不会有任何效果。
指定至少这两个CSS3的动画属性绑定向一个选择器：

1.规定动画的名称；

2.规定动画的时长。

必须定义动画的名称和动画的持续时间。
动画是使元素从一种样式逐渐变化为另一种样式的效果。

可以改变任意多的样式任意多的次数。
用百分比来规定变化发生的时间，或用关键词 “from” 和 “to”，等同于 0% 和 100%。

0% 是动画的开始，100% 是动画的完成。
为了得到最佳的浏览器支持，始终定义 0% 和 100% 选择器。

CSS3 2D 转换

2.学习参考：

https://www.runoob.com/css3/css3-2dtransforms.html

CSS3中matrix详解：https://www.cnblogs.com/seeks/p/7537983.html

3.感悟&总结

CSS3转换可以对元素进行移动、缩放、转动、拉长或拉伸。

2D变换方法有

translate()

translate(Xpx,Ypx)方法，根据左(X轴)和顶部(Y轴)位置给定的参数，从当前元素位置移动。

rotate()

rotate(Xdeg)方法，在一个给定度数顺时针旋转的元素，负值是逆时针旋转角度。

scale()

scale(X，Y)方法，该元素增加或减少的大小，取决于宽度（X轴）和高度（Y轴）的参数：X或者Y大于1则是放大，小于一则是缩小；若为负数时则会倒过来进行放大或缩小。

skew()

skew(30deg,20deg) 元素在X轴和Y轴上倾斜20度30度；参数为负表示向相反方向倾斜。

matrix()

matrix包含一个3 X 3的矩阵，专门用于图像变换匹配。

matrix(a,b,c,d,e,f)有六个参数，包含旋转，缩放，移动（平移）和倾斜功能。

偏移 matrix(1, 0, 0, 1, x偏移量, y偏移量)

缩放 缩小一半，matirx(0.5,0,0,0.5,0,0);

倾斜 水平倾斜30度，cos30°和sin30°的值，作为a和c的值 matrix(0.866,0,0.5,1,0,0)；

旋转 顺时针旋转30度， matrix(0.866,0.5,-0.5,0.866,0,0)；

SVM算法：SVM

SVM可以和kernel trick技术结合使用，利用kernel trick技术进行数据的转换，再根据转换信息在可能的输出之中找到一个最优的边界。SVM就是做很多异常复杂的数据转换工作，接着根据预定义的标签或者输出，进一步计算出如何分离用户需要的数据。

SVM核心思想

一.它针对线性可分情况进行分析，对于线性不可分的情况，通过非线性映射算法使其在高维特征空间线性可分，使用高维特征空间采用线性算法对样本的非线性特征进行线性分析。

二.它基于结构风险最小化理论之上在特征空间中构建最优超平面，使得学习器得到全局最优化，并且在整个样本空间的期望以某个概率满足一定上界。

结构风险最小化理论：把函数集构造为一个函数子集序列,使各个子集按照VC维的大小排列;在 每个子集中寻找最小经验风险,在子集间折中考虑经验风险和置信范围，取得实际风险的最小化。即SRM准则。

VC维用来衡量模型的容量。

VC维：VC维反映了函数集的学习能力，VC维越大则学习机器越复杂（容量越大），是研究学习过程一致收敛的速度和推广性统计学理论定义的有关函数集学习性能的一个重要指标。

定义是：对一个指示函数集，如果存在H个样本能够被函数集中的函数按所有可能的2的H次方种形式分开，则称函数集能够把H个样本打散；函数集的VC维就是它能打散的最大样本数目H。若对任意数目的样本都有函数能将它们打散，则函数集的VC维是无穷大，有界实函数的VC维可以通过用一定的阈值将它转化成指示函数来定义。

尚没有通用的关于任意函数集VC维计算的理论，只对一些特殊的函数集知道其VC维。例如在N维空间中线性分类器和线性实函数的VC维是N+1。

指示函数：在集合论中，指示函数是定义在某集合X上的函数，表示其中有哪些元素属于某一子集A。

SVM（支持向量机）

分类和回归问题都会涉及到SVM算法，SVM就是做很多异常复杂的数据转换工作，接着根据预定义的标签或者输出，进一步计算出如何分离用户需要的数据。

某些对数据的操作是建立在数据是线性可分上进行的，对于某些线性不可分的数据，SVM的核技巧（核方法）把数据转换到更高的维度上，在高维空间上数据可能就会变得线性可分。

核技巧（核方法）：

判断数据是不是线性可分的？

最简单的情况是数据向量是一维二维或者三维的，我们可以把图像画出来，直观上就能看出来。

当唯独变得很高时，检查凸包是否相交。（凸包：在某维度曲线上刚好包住所有数据的凸闭合曲线）

以二维的情况为例，如果我们的数据训练集有两类：M+和M-，当我们画出两个类的凸包，如果两者不重叠，那么两者线性可分，反之则不是线性可分。而且可以用电脑判断凸包M+的边和凸包M-的边是否相交，这样就可以跳过人工绘图的繁琐过程。
